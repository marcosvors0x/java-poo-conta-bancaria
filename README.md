# Sistema de Conta Bancária (POO - Java)

Projeto desenvolvido para praticar Programação Orientada a Objetos em Java.

## Funcionalidades
- Criação de conta com ou sem depósito inicial
- Depósito
- Saque com taxa fixa de R$5
- Encapsulamento e construtores

## Conceitos aplicados
- Classes e objetos
- Encapsulamento
- Construtores
- Sobrecarga
- Método toString

## Tecnologias
- Java
